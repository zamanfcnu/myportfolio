AI / Chatbot

AI Personal Stylist

Python Claude API Prompt Engineering
👗
❓The Problem

People struggle to put together outfits matching their body type, style, and budget — especially for specific occasions. Personal stylists cost £100+/hour and aren't accessible to most people.

💡My Solution

An AI-powered styling assistant that asks 6 targeted questions about user preferences and generates a complete, personalized outfit with specific pieces, color palettes, accessories, and shopping recommendations.

⚙️Process

Researched how personal stylists make decisions → identified key variables (body type, occasion, budget, season, style) → designed conversational UI flow → engineered detailed prompts for Claude API → built structured JSON output → iterated on prompt quality and trend-awareness.

📈Results

Generates personalized outfits in under 5 seconds. Recommendations account for current fashion trends, body-type-specific advice, and budget constraints. Users can regenerate for different options.

🧠What I Learned

Prompt engineering is the most impactful skill when working with AI. Understanding fashion deeply made me better at directing AI than a developer without fashion knowledge.

Data Analytics / AI

Trend Pulse

Python Data Analysis Claude API
📊
❓The Problem

Fashion professionals need to track trend movements for buying, design, and content decisions. Current tools are either too expensive (WGSN costs £5,000+/year) or too surface-level.

💡My Solution

A real-time trend dashboard scoring trends 0-100, tracking momentum, and providing AI-powered analysis with investment recommendations for each trend.

⚙️Process

Identified impactful trends through social media and runway analysis → created scoring system based on social volume, search interest, runway presence → built filterable dashboard → integrated Claude API for on-demand INVEST/SKIP recommendations.

📈Results

Tracks 12+ major trends across 5 categories. AI analysis provides specific advice: how to wear each trend, leading brands, and predicted peak dates.

🧠What I Learned

Data without context is useless. My fashion communication background lets me interpret data in ways pure data analysts can't — I understand WHY trends emerge, not just THAT they're trending.
