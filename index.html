AI / Chatbot

AI Personal Stylist

PythonClaude APIPrompt Engineering
👗
❓The Problem

People struggle to put together outfits matching their body type, style, and budget. Personal stylists cost £100+/hour and aren't accessible to most people.

💡My Solution

An AI-powered styling assistant that asks 6 targeted questions and generates a complete, personalized outfit with specific pieces, color palettes, accessories, and shopping recommendations.

⚙️Process

Researched how personal stylists make decisions → identified key variables → designed conversational UI → engineered prompts for Claude API → built structured JSON output → iterated on quality.

📈Results

Generates personalized outfits in under 5 seconds. Accounts for current trends, body-type advice, and budget constraints.

🧠What I Learned

Prompt engineering is the most impactful skill with AI. Fashion knowledge made me better at directing AI than a developer without fashion context.

Data Analytics / AI

Trend Pulse

PythonData AnalysisClaude API
📊
❓The Problem

Fashion professionals need to track trends. WGSN costs £5,000+/year. Most free tools just count hashtags — no real analysis.

💡My Solution

A real-time dashboard scoring trends 0-100, tracking momentum, with AI-powered INVEST/SKIP recommendations.

⚙️Process

Identified trends via social media + runway analysis → created scoring system → built filterable dashboard → integrated AI for on-demand analysis.

📈Results

Tracks 12+ trends across 5 categories. AI provides how to wear, leading brands, and peak predictions.

🧠What I Learned

Data without context is useless. My fashion background lets me understand WHY trends emerge, not just THAT they're trending.
